# TC-S Network Music System

Music streaming and royalty distribution platform.

## Features

- Song catalog management
- Play event tracking
- Royalty calculation
- Streaming analytics

## Setup

```bash
npm install
cp .env.example .env
npm run dev
```

## API Endpoints

- `GET /api/songs` - List all songs
- `POST /api/play/:songId` - Record play event
